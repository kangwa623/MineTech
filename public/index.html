Post-Blast Hazard Detection Rover

An AI-enabled autonomous rover system for real-time detection of nitrous fumes, toxic gas concentrations, and structural hazards in Zambian mining operations. Zero human exposure during post-blast assessment.

Core Systems

🔬
Gas Detection

Real-time monitoring of NOx, CO, CO₂, and O₂ via precision MEMS sensors. Threshold alerts dispatched via ZigBee mesh within milliseconds of detection.

📡
Mesh Network

ZigBee + WiFi mesh topology provides redundant communication channels. Operates in GPS-denied underground environments without signal loss.

🎯
2D LiDAR SLAM

Hector SLAM algorithm builds real-time occupancy grid maps. Enables precise spatial localisation and gas concentration heat-mapping.

📷
Live Video Feed

Low-latency video stream transmitted to control room. Operators assess blast site conditions and structural integrity without physical presence.

⚠️
Alert Classifier

AI-powered classification of hazard severity — SAFE, CAUTION, DANGER, CRITICAL. Multi-channel alerts to supervisors via mesh and dashboard.

🧭
Obstacle Avoidance

Reactive navigation using LiDAR point-cloud data. VFH+ algorithm enables safe traverse of post-blast debris fields and unstable terrain.

AI Modules

01
// Localisation & Mapping

Hector SLAM

Simultaneous Localisation and Mapping using LiDAR scan-matching. No odometry required — operates reliably on uneven post-blast terrain. Generates occupancy grid maps in real-time at 40Hz.

View SLAM Screen →
02
// Spatial Intelligence

Gas Spatial Heatmap

Overlays gas concentration readings onto the SLAM-generated map. Creates a colour-coded spatial heatmap of hazardous zones, enabling precise no-entry boundary definition.

View Heatmap Screen →
03
// Autonomous Navigation

Obstacle Avoidance

Reactive avoidance using VFH+ (Vector Field Histogram). Processes LiDAR point-cloud at 40Hz to compute safe steering vectors around dynamic obstacles and rubble.

View Avoidance Screen →
04
// Hazard Classification

Alert Classifier

Multi-class hazard severity classifier trained on gas concentration thresholds, sensor fusion data, and historical blast event logs. Issues structured alerts with confidence scores.
